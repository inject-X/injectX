## 💻 应用简介

InjectX 是一款为 macOS 系统设计的应用程序注入工具，支持多种 CPU 架构（arm64 和 x86_64）。

## ✨ 主要特性

- 🚀 简单易用的用户体验
- 🔧 精美优雅的界面设计
- 💡 持续更新优化

## 🛠 系统要求

- macOS 13.0 (Ventura) 或更高版本

## 📒 支持的应用程序

📌 **重要:**  
在使用注入之前，请查看 **[应用列表](AppList.md)** :  
- 支持的应用程序
- 授权信息
- 使用说明

## 📋 快速使用

1. 打开 InjectX 应用
2. 验证注入支持
3. 执行应用程序注入
4. 启动已注入的应用
5. 查看 **[应用列表](AppList.md)** 以获取每个应用的详细说明。
